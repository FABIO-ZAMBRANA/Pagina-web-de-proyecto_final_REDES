Materiales

Equipo de Red

Routers: Se recomienda utilizar modelos como Cisco 1941 o similares, que soporten enrutamiento dinámico y segmentación de red.

Switches: Modelos como Cisco 2960, o equivalentes, para conectar dispositivos dentro de cada subred (Marketing, Ventas, IT).

Servidores: Necesarios para servicios como SNMP, NTP y almacenamiento de datos.


Dispositivos Finales

Computadoras: PC o laptops para cada departamento (por ejemplo, 2 para Marketing, 2 para Ventas, 2 para IT).


Cableado y Conexiones

Cables Ethernet (Cat5e o Cat6): Utilizados para conectar dispositivos a los switches y switches a los routers.

Cables Seriales (DCE/DTE): Necesarios para las conexiones seriales entre los routers, permitiendo la configuración y comunicación entre ellos.
Estos materiales forman la base técnica para configurar una red empresarial robusta y eficiente, que permita la comunicación fluida y segura entre los distintos departamentos y servicios de la organización.
